# Kordal – Videojuego 2D en Unity

## Descripción
Kordal es un videojuego 2D de desplazamiento lateral (side-scrolling) desarrollado en Unity. El jugador controla a un personaje que debe avanzar por niveles lineales, superar obstáculos, evitar enemigos y llegar a la meta. El proyecto forma parte de un trabajo académico orientado a aplicar conocimientos de programación, diseño de videojuegos y organización de proyectos.

## Características principales
- Movimiento lateral del personaje
- Salto, caída y colisiones
- Niveles lineales con inicio y final
- Enemigos básicos con patrones simples
- Sistema de vidas y puntuación
- Interfaz de usuario (UI)
- Música y efectos de sonido

## Estado actual del proyecto
- Player configurado con Rigidbody2D y BoxCollider2D
- Movimiento básico implementado
- Animaciones Idle y Run funcionando
- Suelo con Tilemap y colisiones
- Estructura inicial del proyecto creada

## Tecnologías utilizadas
- Unity 2022+
- C#
- Aseprite (sprites)
- GitHub (control de versiones)

## Objetivo del proyecto
Desarrollar un prototipo funcional que incluya movimiento, animaciones, enemigos, niveles y una interfaz básica, siguiendo una planificación estructurada.
